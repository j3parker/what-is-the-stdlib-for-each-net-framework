# What is the "standard library" for each .NET framework?

MSBuild adds references to some DLLs automatically. This repo was an experiment
to double check that list (vs. figuring it out via msbuild's code).

The stuff below here was generated with this script:

```bash
ls | xargs -n1 -I{} bash -c 'echo "### {}" && echo "\`\`\`" && dotnet build {} --verbosity=diag | grep "/reference:" | sed "s_/reference:_%_g" | tr "%" "\n" | grep "^/" | sed "s_ .*__" | sed "s_.*/__" | sed "s:\.dll$::" && echo "\`\`\`"' > README.md
```
